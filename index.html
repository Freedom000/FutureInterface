Welcome to Paper3D git page.

Paper3D is a cross-platform 3D rendering engine that can greatly shorten the development cycles of live wallpapers and casual games on Android and iOS (coming soon). Compared to other rendering engines, Paper3D provides a full C++ API interface from the top to the bottom layer. Thus it doesn't carry any complexity and instability of mixing programming languages in one engine and provide the maximum performance per watt.

Clone the Paper3D git repo.


$ git clone https://github.com/lihw/FutureInterface.git FutureInterface

Please visit the official website of Paper3D, www.paper3d.net.

Enjoy the journey!
